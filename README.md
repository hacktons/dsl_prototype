# Flutter动态化实验 Fair

Fair是轻量级的有限动态化框架。目前处于实验性质，存在重大变更的可能。

主要用于支持业务的布局动态化。

## 功能
* 支持布局描述动态下发&渲染 
* 支持DSL内联函数&变量
* 扩展其他常用表达式 TODO
* 逻辑动态下发（暂不支持）
* 基于注解快速生成描述文件 TODO
* 开发期间提示不支持的转换情形 TODO
* 配套cli工具链 TODO
* bundle打包服务 TODO
* FlatBuffer格式支持 TODO
